Admins can't access things they should be able to. The role chain is admin → editor → viewer, so admins should inherit all viewer and editor permissions on top of their own. But right now, admins can't read reports (which comes from viewer), can't access the dashboard (also viewer), and somehow even lost the ability to delete users — which is explicitly defined on the admin role itself. Viewer and editor users are fine; it's only admin that's broken.
